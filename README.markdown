##Easily make text output to the terminal in color

###Usage example:

	puts "my string".red

Available colors

 - gray
 - red
 - green
 - yellow
 - blue
 - purple

###Steps
 
 1. `$> gem install color_text` *#Install the gem*
 2. `require 'color_text'` *#Include the gem*
 3. **Your ready to start using.**
